# 无限画布桌面版安装指南

## Windows

1. 下载 Windows `.exe` 安装包并双击运行。
2. 在安装向导中选择安装范围和程序安装目录。
3. 安装完成后保持“运行无限画布”勾选，进入首次启动向导。
4. 确认本地数据目录。建议放在空间充足的本地磁盘，并至少预留 10 GB。
5. 等待应用自动准备 Python 和后端依赖。
6. 按需安装 CLI 工具；这些工具是可选项，可以跳过后在设置中配置。
7. 初始化完成后自动进入主界面。

安装包尚未进行代码签名时，Windows 可能显示 SmartScreen 提示。应先核对安装包来源和版本，再通过“更多信息 → 仍要运行”继续。

## macOS

1. 下载与芯片架构匹配的 `.dmg`：Apple Silicon 使用 arm64，Intel 使用 x64。
2. 打开 `.dmg`，将“无限画布”拖入“应用程序”。
3. 从“应用程序”打开无限画布。
4. 按首次启动向导确认数据目录并等待运行环境准备完成。

未签名版本可能被系统拦截。确认安装包来源后，可在“系统设置 → 隐私与安全性”中允许打开。

## 本地数据

首次启动会显示实际数据目录。目录内包含：

- 项目、画布、会话和设置
- 上传的素材
- 生成的图片和视频
- 导出结果和自定义工作流

升级程序不会覆盖此目录，卸载程序也不会自动删除它。需要彻底清理时，请先确认不再需要其中的项目和媒体，再手动删除。

建议定期备份整个数据目录。迁移到新电脑时，先复制完整目录，再在首次启动向导中选择该目录。

## 首次启动需要网络的内容

当前安装包会在首次启动时自动准备 Python，并通过镜像安装 Python 依赖；安装部分可选 CLI 工具也需要网络。完成初始化后，本地项目管理和本地文件访问可以离线使用，但调用在线生成模型仍需要网络。

## 常见问题

### 数据目录不可写

选择当前用户有写入权限的目录。不要选择程序安装目录、系统目录、只读移动介质或受企业策略保护的目录。

### 空间不足

图片和视频会持续占用空间。更换到空间充足的磁盘，或备份后清理不再使用的生成结果。建议至少保留 10 GB，视频工作流建议预留更多。

### Python 或依赖安装失败

先检查网络和代理设置，然后在错误窗口选择“重试”。应用使用独立运行环境，不要求用户手动修改系统 Python。

### 升级后数据不见了

不要新建空目录。重新选择之前的数据目录；如果不确定位置，可查找 `storage-state.json` 中记录的 `root`。

### 如何完整卸载

先通过系统卸载应用。默认会保留本地数据，避免误删项目；确认已备份且不再需要后，再手动删除首次启动向导中显示的数据目录。
